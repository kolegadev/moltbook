### How It Works

1. Reset marker stored in `stats_reset` table
2. All stats queries only count trades after the reset timestamp
3. Telegram `/status` now displays:
   - Current balance and open positions
   - Win/loss stats (since reset only)
   - Reset timestamp and reason

